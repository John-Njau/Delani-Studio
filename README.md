# Installation

To Run this app in your machine, you need to:

- Open Terminal `Ctrl+Alt+T`

- git clone `https://github.com/John-Njau/Delani-Studio.git`

- `cd Delani-Studio`

- `code . or atom .` based on the text editor you have to open and view the project.
